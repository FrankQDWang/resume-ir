# 安全、性能与验收指标

## 1. 安全原则

简历数据包含姓名、电话、邮箱、教育、工作经历等个人信息。系统默认策略：

1. 数据不出本机。
2. 联系方式默认脱敏展示。
3. 联系方式索引用 hash 或加密字段。
4. 日志禁止写入完整手机号、邮箱、身份证、原文大段内容。
5. OCR 中间文件默认加密或任务结束后清理。
6. 诊断包默认脱敏。
7. 删除文件后，索引和缓存必须进入删除流程。

## 2. 本地数据目录

逻辑目录建议：

```text
app_data/
  meta/          # 元数据、任务、状态、配置
  fts/           # 倒排索引快照
  fields/        # 字段列快照，可与倒排同存储
  vector/        # 向量索引快照
  cache/         # OCR、snippet、embedding 缓存
  temp/          # 临时文件，崩溃后可清理
  logs/          # 脱敏日志
  backups/       # 元数据备份和索引 manifest
```

## 3. 删除语义

删除分为三种：

| 类型 | 行为 |
|---|---|
| 从索引移除 | 查询不可见，原文件不动 |
| 删除缓存 | 删除 OCR、snippet、embedding 中间缓存 |
| 彻底清理 | 延迟合并或重建索引，移除旧 segment 中的残留 |

必须向用户说明：某些索引格式的物理清理需要合并或重建，不一定在点击删除瞬间完成。

## 4. 性能指标

### 4.1 查询指标

| 指标 | 目标 |
|---|---:|
| 文件名查询 P95 | `<50ms` |
| 全文查询 P95 | `<100ms` |
| 语义查询 P95 | `<150ms` |
| 混合查询 P95 | `<200ms` |
| 查询 P99 | 不超过 P95 的 3 倍 |
| 结果组装 top20 | `<30ms` |

### 4.2 导入指标

导入指标按文件类型分开：

| 类型 | 指标 |
|---|---|
| docx 文本简历 | 每核每秒处理数、失败率 |
| 文本层 PDF | 每页抽取耗时、乱码率 |
| 扫描 PDF | 每页 OCR 耗时、OCR 置信度 |
| 字段抽取 | F1、耗时、冲突率 |
| 向量生成 | docs/s、batch 延迟、内存峰值 |

### 4.3 资源指标

| 指标 | 说明 |
|---|---|
| RSS | 常驻内存 |
| 峰值 RSS | 导入和 OCR 时峰值 |
| CPU 平均/峰值 | 后台任务限流依据 |
| 磁盘读写量 | 写放大和索引合并评估 |
| 索引大小/原文大小 | 容量规划 |
| 冷启动时间 | 启动到可查询 |
| 热启动时间 | 已有快照打开耗时 |

## 5. 质量指标

| 能力 | 指标 |
|---|---|
| 全文检索 | Recall@K、MRR、NDCG |
| 语义检索 | Recall@K、人工相关性评估 |
| 字段抽取 | precision、recall、F1 |
| 去重 | pair precision、pair recall |
| OCR | 字符错误率、字段可抽取率 |
| 混合检索 | 业务 query top10 命中率 |

## 6. 基准数据集

至少准备：

1. 10 万份混合格式简历，用于日常性能回归。
2. 100 万份压测数据，用于容量和尾延迟测试。
3. 1,000 份人工标注字段样本，用于抽取质量。
4. 500 条真实业务查询，用于检索质量。
5. 200 份扫描件样本，用于 OCR 质量。
6. 100 个损坏、加密、异常文件，用于边界测试。

所有基准数据必须脱敏。

## 7. 验收门槛

第一版生产可用验收：

| 项 | 门槛 |
|---|---|
| 10 万简历导入 | 不崩溃，可断点续跑 |
| 100 万简历索引 | 可构建、可查询、可重启恢复 |
| 查询 | 混合查询热状态 P95 接近目标 |
| OCR | 只对需要 OCR 的文件触发，可暂停可恢复 |
| 字段 | 联系方式、学历、技能、公司、职位、时间段可用 |
| 删除 | 删除后默认不可查，后台可清理 |
| 崩溃恢复 | 随机 kill 后不破坏已提交快照 |
| 低配机器 | Economy 档不持续占满 CPU/内存 |
| 跨平台 | Windows/macOS 均能导入、查询、更新、卸载 |

## 8. 观测面板

本地诊断至少包含：

1. 索引文档数。
2. 可查询文档数。
3. partial 文档数。
4. 各任务队列深度。
5. 最近错误 TopN。
6. 查询 P50/P95/P99。
7. CPU/内存/磁盘趋势。
8. 当前性能档位。
9. 索引快照版本。
10. 数据目录占用。
