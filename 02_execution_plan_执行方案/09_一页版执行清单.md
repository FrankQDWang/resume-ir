# 一页版执行清单

## 先做

1. Rust daemon + CLI + 本地 IPC。
2. SQLite/SQLCipher 元数据和任务队列。
3. docx/PDF 文本层解析。
4. Tantivy 文件名/全文/字段索引。
5. 规则字段抽取。
6. HNSWlib + ONNX embedding。
7. RRF 混合检索。
8. OCR worker 只处理需要 OCR 的扫描 PDF。
9. Windows/macOS 安装包。
10. 10 万/100 万压测和故障注入。

## 不先做

1. Elasticsearch/Milvus/Weaviate 本地服务化主内核。
2. 查询时 OCR。
3. 查询时长文本大模型抽取。
4. Electron 重壳。
5. 全量 PDF 无脑 OCR。
6. 没有置信度的字段覆盖。
7. 没有快照回滚的索引写入。

## 最重要的门禁

1. 查询 P95 不因导入和 OCR 被拖垮。
2. 低配机器 Economy 档不持续占满 CPU。
3. 崩溃不破坏已提交快照。
4. 日志和诊断包不泄露明文 PII。
5. Windows/macOS 每个 PR 至少 smoke build。
